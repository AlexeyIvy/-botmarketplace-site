# Стратегия 3: Funding Rate Arbitrage + Delta Hedge

> **Приоритет:** 🥉 Флагман #3 — рыночно-нейтральная стратегия  
> **Аудитория:** Консервативные инвесторы  
> **Edge:** 19.26% годовых при max drawdown 0.85% — не зависит от направления рынка

## Концепция

Стратегия использует ставку финансирования (Funding Rate) бессрочных фьючерсов. При положительной ставке лонги платят шортам каждые 8 часов. Алгоритм открывает **лонг на споте** и **шорт на фьючерсе** с одинаковым объёмом — delta-neutral позицию — и получает выплату ставки финансирования без риска направления рынка.

## Доказанные метрики

| Метрика | Значение |
|---|---|
| Годовая доходность | **19.26% (2025)** |
| Max Drawdown | **0.85%** |
| Sharpe Ratio | **5.0+** |
| Beta к рынку | **~0.135** (почти нейтральный) |
| Зависимость от направления | Минимальная |
| Лучший рынок | Любой (особенно бычий — ставки выше) |

## Логика стратегии

```
Шаг 1: Сканировать Funding Rate по всем символам
   → Отбор пар, где Funding Rate > порогового значения (0.01% per 8h)
   → Расчёт ожидаемой годовой доходности: rate × 3 × 365

Шаг 2: Открыть Delta-Neutral позицию
   → Spot Long: купить X USDT актива на споте
   → Perp Short: открыть шорт X USDT того же актива на фьючерсах
   → Размер: одинаковый на обеих ногах → delta = 0

Шаг 3: Получать Funding Rate каждые 8 часов
   → Время выплат Bybit: 00:00 / 08:00 / 16:00 UTC
   → Прибыль = Funding Rate × размер позиции × 3 раза в день

Шаг 4: Мониторинг и выход
   → Выход: Funding Rate опускается ниже порога рентабельности
   → Выход: Резкое расхождение спота и фьючерса (basis risk)
   → Выход: Ставка фиксированная на весь период или динамическая

Шаг 5: Ротация капитала
   → Автоматически переходим в пару с более высокой ставкой
   → Диверсификация по 5–15 символам одновременно
```

## Параметры стратегии

| Параметр | Значение |
|---|---|
| Тип | Spot Long + Perpetual Short (delta-neutral) |
| Порог входа | Funding Rate > **0.01%** per 8h (≈ 10.95% годовых) |
| Порог выхода | Funding Rate < **0.005%** per 8h |
| Плечо фьючерса | **1–2x** (минимальное для снижения liquidation risk) |
| Диверсификация | До **15 пар** одновременно |
| Лучшие символы | BTC, ETH, SOL, BNB (высокий open interest) |
| Время выплат | 00:00 / 08:00 / 16:00 UTC (Bybit) |
| Капитал на символ | 5–10% от портфеля |
| Мин. капитал | ~$5,000 USDT (для разумной диверсификации) |

## Риски и митигация

| Риск | Описание | Митигация |
|---|---|---|
| Basis Risk | Спот и фьючерс расходятся в цене | Мониторинг спреда, выход при расхождении > 0.5% |
| Liquidation Risk | Фьючерс ликвидируется при резком движении | Плечо 1–2x, буфер маржи 50% |
| Negative Funding | Ставка становится отрицательной | Автовыход при ставке < 0 |
| Exchange Risk | Риск биржи (hack, insolvency) | Диверсификация по биржам в prod |

## Примечание для MVP

В MVP (demo) Bybit позволяет торговать только `linear` (USDT perpetual). Спот-нога требует подключения **Bybit Spot API** — отдельного эндпоинта. Это расширение за рамками текущего MVP, но стратегия должна быть заложена в архитектуру с Phase 1 в части data model.
